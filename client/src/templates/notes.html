Besides giving information on the competition (which you can find easily anywhere on the web), this site is mainly a technological prototype. The project is hosted on github:

  • https://github.com/sfidanza/worldcup

Main technical features:

  • javascript MVC with back/forward/reload support
  • json data API
  • server backend using nginx, node.js and MongoDB
  • running in Docker container and deployed on Docker Swarm
  • https support with grade A+ security (handled by a Traefik gateway)
  • responsive design to support different sizes from mobile to desktop versions

Functional features:

  • List of matches, locations, date/hours (France time), with groups schedule and results
  • Finals board with group tooltips on round of 16 (mouseover/tap to display)
  • Highlight team matches in group (mouseover/tap on team ranking)
  • Bet on the final winner (register/login on the site, or login with your google account)
  • Contains multiple worldcup editions: check the History to switch between editions
  • Contains multiple UEFA Euro editions and the latest Club World Cup format
  • Calendar integration (.ics) during live competitions to get all matches and results directly in your calendar
  • Live score during matches (using Server-sent events) for a lightweight follow up when you can't watch the game
