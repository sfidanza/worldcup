Besides giving information on the competition (which you can find easily anywhere on the web), this site is mainly a technological prototype. It demonstrates what can be done using client-side templates, with JSON client-server communication.

The project is hosted on github, and you will find details about the technical features there, as well as analysis and pointers on the toolchain and development setup (see README and wiki entries):

  • https://github.com/sfidanza/worldcup2014

Summary of client-side technical features:

  • All displays are client-side templates
  • Data is retrieved on load from the server
  • Supported browsers: FF, Chrome, IE8+, IE7 (with minor glitches - but who cares?) - (Opera, Safari not tested)
  • Back/Forward is supported (in FF, Chrome, IE8+)
  • Reload restores the current state

Functional features:

  • "Finals board", with group tooltips on quarter finals
  • Highlight team matches in schedule on mouseover in team rankings (not in IE7)

Ideas for improvement:

  • Mobile-specific display
  • Support localization
  • Betting engine (group leads, winners in finals board, final winner)
